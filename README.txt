Kirby's Identity Crisis

Thomas Huang and Kaitlyn McIntyre


Game Controls:
	Move- Arrow keys
	Jump- Up arrow
	Make clone- Space or E
	Cycle between clones- Tab or Q
	Restart Level- R
	Pause Menu- Escape key

Along with the controls, the game has a tutorial that shows how to play.
Accessing the pause menu allows the player to return to the main menu and also to jump directly to level select (in case the current level is too hard)
